# PDT App MVP

React Native Expo app for health tracking dashboard with Firebase phone OTP auth.

## Setup
1. `npm install`
2. `npx expo install --fix`
3. `npx expo doctor`
4. Update src/services/firebase/config.ts with your Firebase config
5. `npx expo start`

## Features
- Phone OTP login (Firebase/mock)
- Profile setup
- Dashboard with mock health metrics

UI follows provided images.
