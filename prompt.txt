OTP Verification Screen
Auth Step 2
Layout
Same background as welcome. Bottom card slides up from y+100 with spring. Avatar fades to 30% opacity behind card (depth effect).
Card has rounded top corners 24px. "Verify Your Number" title 22px/500 weight. Phone number in muted gray below.
OTP Input Boxes
6 equal boxes in a flex row with gap: 10px. Each box: 48×56px, 12px radius, border 1.5px solid #E2EEFF idle → #2962FF active → #00C853 filled.
Active box: scale(1.06) with border glow ring rgba(41,98,255,0.15). Each box animates individually — on digit entry, number pops in with scale(1.3→1.0) 150ms.
On all 6 filled: boxes briefly flash green border simultaneously (pulse 200ms), then "Verify & Continue" button enables and glows.
Auto-advance focus to next box on digit input. Backspace returns focus to previous.
Countdown & Resend
Circular countdown arc (thin 2px stroke, 40px diameter) wraps the "Xs" timer in blue. Arc depletes over 60s.
Resend becomes tappable at 0s — text color transitions from #AAAAAA → #2962FF with 300ms color animation.
Verify Button
Same styling as Get OTP. Disabled state: #B3CCFF. On success → button morphs to a green checkmark with circular expand animation (ripple outward).
"Change number" link: 14px, #888, center-aligned, 44px tap target height.

Header

Header has no background on scroll-top. On scroll-down 48px → frosted glass blur bg appears with 200ms transition.

On screen enter: arcs animate from 0 to fill value over 800ms with ease-out curve. Score number counts up from 0 to 75 in 600ms.
Legend items right of ring — dot + metric/goal in bold + label muted gray, 14px.

3-column equal grid of mini cards. Each: white card, icon (24px colored), value 20px/500, label 12px muted, progress bar 4px height with colored fill, percentage 11px right-aligned.
Progress bars animate from 0% to value on entry (600ms ease-out). Bar radius: pill shape 99px.
Cards have a subtle press state: scale(0.97) with haptic feedback on tap.
Health Metrics Grid
2-column grid. Each card: icon top-left (32px colored), large value 28px/500, label 13px muted, thin colored indicator bar at bottom (not full progress — just a brand stripe).
Skeleton shimmer loaders shown for 800ms before data populates. Shimmer: light gray (#F0F2F5) sweep animation left to right.
Heart Rate card: 81 bpm in bold red, small sparkline line chart below (6 data points, 2px stroke, no fill). Blood Oxygen: 98% in teal.
Sleep & Weight cards same layout pattern. Add unit labels (hrs, kg) as small superscript-style text.
AI Insight FAB
Floating pill button bottom-left: "AI Insight" with spark icon. 48px height, #2962FF bg, white text, 99px radius. Subtle bounce animation on load entry (translateY 0→-6px→0, 400ms).
Bottom Nav
3 tabs: Health, Workouts, Me. Active tab: filled colored icon + label in #2962FF. Inactive: #AAAAAA. Active indicator: small 4px wide pill underline or filled bg pill 32px wide behind icon. Tab switch: icon scale 1.0→1.15→1.0 with 150ms spring.
