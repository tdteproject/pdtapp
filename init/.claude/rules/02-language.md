# Language Policy

All ATESCHH KIT system content is in English. This includes:

- Rules, workflows, agent definitions, skill definitions, templates
- Generated artefacts (STATE.md, PLAN.md, REQUIREMENTS.md, DESIGN.md, WIREFRAMES.md, STRUCTURE.md, DECISIONS.md, BACKLOG.md, MEMORY.md, SESSION-LOG.md, ACTIVE-PROJECT.md, ACTIVE-CONTEXT.md)
- Internal logs, agent prompts, agent return summaries, MemPalace entries
- Code identifiers, filenames, commit messages

The user's chat language is independent of this rule. Reply to the user in their language; write to disk in English.

Tone: concise, direct, professional. No filler.
